## 🚀 Featured Projects

### 🤖 [Self-Balancing PID Platform](https://github.com/Sam-kdk/Self-Balancing-PID-Platform)

> A real-time feedback control system that keeps a ball balanced on a tilting platform using a PID controller, HC-SR04 ultrasonic sensor, and servo motor — all running on Arduino.

- ⚙️ Built with: `C++` `Arduino` `PID Control` `HC-SR04` `Servo Motor`
- 📐 Implements full PID algorithm with anti-windup integral protection
- 📊 Live Serial Monitor output for real-time PID tuning and debugging
- 🎯 Achieves < 2mm steady-state error from setpoint
